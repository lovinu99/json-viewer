Features & How to Use

  • Edit JSON: Paste or type raw JSON on the left. Syntax is automatically highlighted.
  • Format JSON: Prettifies your raw JSON code with proper indentation.
  • Generate Grid: Renders your nested JSON arrays and objects into easy-to-read tables.
  • Hide Fields/Rows: Click the table headers to hide columns. Click the index number to hide rows. Click the chips above the grid to restore them.
  • Trim Fields/Rows: Trim hidden fields/rows by clicking [✂️ Trim JSON].
  • Zoom: Use Ctrl + Mouse Wheel to independently scale text.
  • Resize Panels: Drag the middle vertical bar to adjust workspace proportions.
  • History: Use Ctrl + Z and Ctrl + Shift + Z to undo/redo.
